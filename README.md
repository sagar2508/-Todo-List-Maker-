# -Todo-List-Maker-
📝 Todo List Maker
A simple and user-friendly Todo List application built using Python and Flask that helps users manage daily tasks efficiently. The application allows users to add, update, and delete tasks with ease.
🚀 Features
Add new tasks
Update/Edit tasks
Mark tasks as completed
Delete tasks
Simple and clean user interface
🛠️ Tech Stack
Backend: Python, Flask
Frontend: HTML, CSS
Database: SQLite
📂 Project Structure
Todo-List-Maker/
│── app.py
│── templates/
│── static/
│── database.db
│── requirements.txt
⚙️ Installation & Setup
Clone the repository:
git clone https://github.com/sagar2508/-Todo-List-Maker-.git
Navigate to the project directory:
cd -Todo-List-Maker-
Create a virtual environment:
python -m venv venv
Activate the environment:
venv\Scripts\activate
Install dependencies:
pip install -r requirements.txt
Run the application:
python app.py
▶️ Usage
Open the application in your browser
Add your daily tasks
Mark tasks as completed
Edit or delete tasks anytime
